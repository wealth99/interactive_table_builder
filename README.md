# Interactive Table Builder

문서의 표를 HTML로 변환하고 싶은데, 매번 손으로 칠 수 없으니까 만들었음. 생성 후 수정도 가능.

## 뭐가 되냐고?

- **테이블 변환**: 탭으로 구분된 텍스트 → HTML 테이블 자동 생성
- **셀 편집**: 클릭해서 텍스트, colspan, rowspan 수정 가능
- **클래스 관리**: 행별/열별로 클래스 일괄 적용 (또는 제외)
- **다중 셀 병합**: Shift+클릭으로 여러 셀 한 번에 병합
- **일괄 변경**: 특정 행/열을 TD→TH로 바꾸거나 속성 추가
- **코드 복사**: 생성된 HTML을 클립보드로 복사
